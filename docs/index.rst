Welcome to CPGE Tools Module's documentation!
===================================

**cpgetools** is a Python library specifically crafted for CPGE (Classes Préparatoires aux Grandes Écoles) students, aiming to enhance their understanding of computer science data structures. 

This library empowers users to effortlessly visualize complex structures such as trees, heaps, graphs, automata, and more, directly within their Python code. 

By seamlessly integrating graphical representations into the coding environment, cpgetools facilitates a clearer comprehension of intricate concepts, making it an invaluable resource for CPGE students navigating the intricacies of computer science studies.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
